# 🌱 K Group of Nursery

A modern and responsive website developed for **Shree Chintamani Nursery**, focused on promoting Net Pot Tray solutions and plantation services.

---

## 🚀 About the Project

This project represents a digital platform for a nursery business that specializes in:

- 🌱 Net Pot Tray Supply  
- 🌼 Tecoma Plantation (Bicolor, Yellow, Orange)  
- 🌿 Kamini Plantation  

The website is designed to showcase services, attract customers, and provide contact details in a clean and professional way.
